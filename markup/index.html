Перегляд історії

Команди для перегляду історії

  • $ git log
  • $ git log --follow [файл]
  • $ git diff [первая ветка]...[вторая ветка]
  • $ git show [коммит]

Основні Команди

  • $ git status
  • $ git add.
  • $ git commit -m ""
  • $ git push

Команда для забору файлів із сайта git

  • $ git pull
